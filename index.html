Projects

Cloud-Native CI/CD Pipeline with Kubernetes Deployment

Designed and implemented a production-grade CI/CD pipeline using Jenkins integrated with GitHub, Maven, SonarQube, and Docker to automate the entire software delivery lifecycle. The pipeline included automated code checkout, dependency resolution, static code analysis, unit testing, artifact packaging, container image creation, and image scanning.

Container images were pushed to AWS ECR and deployed to Amazon EKS using Kubernetes Deployments, Services, Ingress Controllers, and ConfigMaps. Infrastructure was provisioned using Terraform with reusable modules for VPC, EKS, IAM roles, security groups, and load balancers.

Implemented rolling updates, readiness/liveness probes, and Horizontal Pod Autoscaling to ensure zero-downtime deployments. Monitoring and alerting were integrated using Prometheus and Grafana dashboards to track CPU usage, memory, request latency, and pod health.

Automated Auto Scaling Monitoring & Configuration System

Built an automated monitoring and configuration system for AWS Auto Scaling Groups to dynamically manage EC2 instances during scale-out and scale-in events. Integrated AWS Lambda functions triggered via CloudWatch Events to detect instance lifecycle changes.

Used Ansible to automatically configure new EC2 instances with required application dependencies, security configurations, and monitoring agents. This eliminated manual provisioning and ensured configuration consistency across environments.

Implemented centralized logging and monitoring using Prometheus, Grafana, and CloudWatch metrics to provide real-time operational visibility. Designed alerting policies to reduce incident response time and improve system reliability.

Secure Three-Tier Music Streaming Platform (AWS)

Developed a scalable three-tier cloud architecture consisting of EC2 instances for the application layer, Amazon RDS (MySQL) for the database layer, and Amazon S3 for object storage of media files. Designed a secure VPC architecture with public and private subnets, NAT Gateway, route tables, and security groups.

Implemented IAM role-based access control for secure service-to-service communication and configured Application Load Balancer for traffic distribution. Enabled Auto Scaling Groups to ensure high availability and horizontal scalability under variable load.

Integrated CI/CD pipelines for automated deployment of backend and frontend services. Configured CloudWatch for monitoring instance health, database metrics, and application logs. The architecture ensured fault tolerance, improved security posture, and optimized infrastructure cost.
